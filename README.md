# Webes technológiák áttekintése

Ez a markdown fájl egy rövid feladatleírást és a leggyakrabban használt webes technológiák bemutatását tartalmazza.
A cél, hogy alapfogalmakat, jellemzőket és egyszerű példákat mutasson be.

A feladat során a technológiák rövid felsorolása után részletesebb kifejtés, majd mintakódok is szerepelnek.

## Felhasznált technológiák

* GIT
* HTML
* CSS3
* JavaScript

---

**GIT**: Egy elosztott verziókezelő rendszer, amely lehetővé teszi a forráskód változásainak követését.
Segítségével több fejlesztő is dolgozhat ugyanazon a projekten anélkül, hogy felülírnák egymás munkáját.
Ágak (branch-ek) és commitok segítségével jól kezelhető a fejlesztési folyamat és a hibajavítás.

```bash
git init
git add .
git commit -m "Első commit"
```

**HTML**: Egy leíró nyelv, amelyet a böngésző értelmez, és a benne található információk alapján megjelenít egy weboldalt.
Az oldal felépítését elemek és úgynevezett TAG-ek határozzák meg.
A HTML biztosítja a tartalom logikai szerkezetét, például címsorokat, bekezdéseket és listákat.

```html
<!DOCTYPE html>
<html>
  <head><title>Példa</title></head>
  <body><h1>Hello világ!</h1></body>
</html>
```

**CSS3**: Egy stílusleíró nyelv, amely a HTML elemek megjelenéséért felel.
Segítségével beállítható a színezés, betűtípus, elrendezés és az animációk is.
A CSS3 modern funkciói reszponzív, különböző eszközökön jól működő felületeket tesznek lehetővé.

```css
h1 {
  color: blue;
  font-size: 2rem;
}
```

**JavaScript**: Egy programozási nyelv, amely a weboldalak interaktivitását biztosítja.
Eseményekre reagál, módosítja a DOM-ot és képes adatokat kezelni vagy szerverrel kommunikálni.
A modern JavaScript böngészőben és szerveroldalon (Node.js) is használható.

```javascript
document.querySelector('h1').addEventListener('click', () => {
  alert('Kattintás történt!');
});
```
